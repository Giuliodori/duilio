# Contributing

Thanks for your interest in contributing to Duilio!

At this stage, contributions are mainly handled through GitHub Issues.

## How to contribute
- Use **GitHub Issues** for bug reports, questions, and feature requests.
- Before proposing major changes, please open an issue to discuss them first.

## Hardware-related issues
Please include:
- Board revision
- Photos (top and bottom)
- Power inputs used
- Wiring description
- Connected motor drivers and loads

## Firmware / software-related issues
Please include:
- Logs or error messages
- Steps to reproduce the issue
- Control method used (RC, USB, RS485, Raspberry Pi)
- Host system details (PC, Raspberry Pi model, OS)

Pull requests are welcome, but please open an issue first to discuss the proposed changes.

More detailed guidelines will be added as the project grows.
